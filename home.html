Ares

A simple slash commands moderation discord bot.



Description

Ares is a simple moderation bot capable of running many commands. Some commands include, /ban, /unban, /kick, /mute, /nickname, etc. The bot also uses a log system to log when a user joins your server, leaves your server, deletes or edits a message, and of couse all the commands used by admins.
